# Undertale-World
This is a dimensional exploration mod for Undertale. Although my personal technical skills are limited, I strive to recreate and expand the world of Undertale in Minecraft in my own way.
 
Find a small hill in the Overworld. You need to use a Magic Stone — crafted from diamonds, bottles o' enchanting, and three emeralds — to activate the bedrock portal hidden inside the hill, which will take you to a mysterious Underground world.
 
Version 0.1 is very rudimentary, yet it already features two new biomes, a full set of new equipment, several new materials, four new generated structures, and a boss worthy of your challenge — Undyne — waiting for you to discover.
